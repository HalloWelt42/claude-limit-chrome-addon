So funktioniert's

Diese Extension ruft deine Verbrauchsdaten direkt von der Claude.ai API ab. Du musst eingeloggt sein auf claude.ai – es wird kein API Key benötigt.

Die Daten werden alle 5 Minuten automatisch synchronisiert. Du kannst auch manuell im Popup auf das Sync-Icon klicken.

Voraussetzungen

  • Claude Pro oder Max Abo
  • Eingeloggt auf claude.ai
  • Cookies für claude.ai erlaubt
